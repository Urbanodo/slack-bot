# MyBot — Slack Bot 🤖

Bot Slack construit avec Node.js et Bolt.js.

## Fonctionnalités

- Répond à la commande `/hello`
- Répond à la commande `/help`
- Répond aux mentions `@MyBot`
- Log tous les messages du canal

## Installation

```bash
npm install
```
